# Food Delivery Web Application

## Project Setup Steps

1. Download or clone the project.
2. Open the project folder in Visual Studio Code.
3. Install the Live Server extension.
4. Open the file named `index.html` (or `FOOD DELIVERY APP.html` if you have not renamed it).
5. Right-click the HTML file and choose **Open with Live Server**.

## Features Implemented

- Responsive user interface
- Navigation bar
- Hero section
- Search functionality
- Food categories
- Restaurant cards
- Menu display
- Add to Cart button (UI)
- Footer
- Mobile-friendly layout

## Data Structures Used

- **Arrays** – to store restaurant and menu data.
- **Objects** – to store restaurant details and food item information.
- **JSON** – data stored in `product.json`.
